# Scalpel

Precise, atomic file editing for code and text over MCP.

## Status

Early project bootstrap. The initial design lives in [SPEC.md](./SPEC.md).

## Direction

- TypeScript
- Official Model Context Protocol TypeScript SDK
- `stdio` transport for local coding clients

## Planned focus

- Exact-string and line-oriented edits
- Dry-run diffs before mutation
- Atomic multi-edit operations
- Strict validation with clear machine-readable errors
